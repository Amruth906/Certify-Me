# Micro-Certification Platform

A simple web app to take quizzes and get certificates.  
Made with **React**, **Node.js/Express**, and **MongoDB**.

## Features
- User Register & Login
- Take Quizzes
- Instant Score
- Download Certificate
- View Quiz History
